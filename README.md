# Test Tiendamia - API

## Requirements

- NodeJS v.18.16.0 on your operating system.

## Installation

```bash
$ npm install
$ npm run migration:run
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
